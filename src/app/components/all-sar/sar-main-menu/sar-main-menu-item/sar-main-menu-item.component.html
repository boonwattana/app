ตอนที่1 ข้อมูลส่วนตัว
1.1 ข้อมูลทั่วไป
  • ข้อมูลทั่วไป
  • ข้อมูลการลาประจำปี
    • 1.2 ข้อมูลการปฏิบัติหน้าที่
  • ข้อมูล ปฏิบัติการสอนตลอดปีการศึกษา
  • ข้อมูล กิจกรรมพัฒนาผู้เรียนที่ปฏิบัติการพัฒนาผู้เรียนตลอดปีการศึกษา
  • ข้อมูล ปฏิบัติหน้าที่ครูที่ปรึกษาตลอดปีการศึกษา
  • ข้อมูล งานพิเศษ ในโรงเรียน/การปฏิบัติหน้าที่อื่น ๆ ที่ได้รับมอบหมาย
    • 1.3 การจัดกิจกรรมการเรียนการสอน
  • จัดทำแผนการจัดการเรียนรู้
  • ผลิตสื่อ / นวัตกรรม
  • จัดทำหน่วยการเรียนรู้แบบบูรณาการ (เศรษฐกิจพอเพียง,ศาสตร์พระราช,สวนพฤกษศาสตร์)
  • จัดทำวิจัยในชั้นเรียน
  • การนำ/มอบหมายงาน นักเรียนไปศึกษาค้นคว้า/ใช้แหล่งเรียนรู้นอกโรงเรียน
  • เชิญวิทยากรภายนอกมาให้ความรู้แก่นักเรียน
  • รูปแบบ / วิธีการจัดกิจกรรมการเรียนการสอนที่ครูใช้
  • สภาพการปฏิบัติงานสอน
  • การพัฒนาตนเอง
  • การได้รับรางวัล / ประกาศเกียรติคุณ / ผลงานดีเด่น / เกียรติประวัติที่ปรากฏต่อสาธารณชนด้านสถานศึกษา / ครู / นักเรียน
  • การได้รับเชิญเป็นวิทยากร/กรรมการตัดสินภายในและภายนอกสถานศึกษา
    • 1.4 ผลการปฏิบัติงาน
  • การปฏิบัติหน้าที่จัดกิจกรรมการเรียนการสอนประจำปีการศึกษา
  • การปฏิบัติงานหน้าที่พิเศษ
    • 1.5 ผลการประเมินการสอนของครูโดยนักเรียน
  • ผลการประเมินการสอนของครูโดยนักเรียน
    • 1.6 การประเมินตนเองเกี่ยวกับการจัดทำแผนการจัดการเรียนรู้ที่เน้นผู้เรียนเป็นสำคัญ
  • การประเมินตนเองเกี่ยวกับการจัดทำแผนการจัดการเรียนรู้ที่เน้นผู้เรียนเป็นสำคัญ
    • 1.7 ผลการประเมินผู้เรียน
  • ผลการประเมินการอ่านคิด วิเคราะห์และเขียน
  • ผลการประเมินคุณลักษณะอันพึงประสงค์
    • ตอนที่2 ผลการดำเนินงานตามมาตรฐานการศึกษา
  • มาตรฐานที่1 คุณภาพของผู้เรียน
  • มาตรฐานที่2 กระบวนการบริหารและการจัดการ
  • มาตรฐานที่3 กระบวนการจัดการเรียนการสอนที่เน้นผู้เรียนเป็นสำคัญ
    • รายงานผลการปฏิบัติงาน และผลการประเมินตนเองรายบุคคล
  • อัพโหลดภาคผนวก
  • รายงาน คำสั่งแต่งตั้งการปฏิบัติหน้าที่
  • สร้าง รายงานผลการปฏิบัติงาน และผลการประเมินตนเองรายบุคคล

    • *หมายเหตุ: กรุณากรอกข้อมูลด้านบนให้ครบถ้วนก่อนสร้างรายงาน*
